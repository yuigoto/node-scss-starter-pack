SCSS STARTER PACK

Olá, seja bem-vindo ao Starter Pack SCSS!

Se você está vendo esta tela, está tudo certo para que possa desenvolver seu projeto usando SCSS e HTML!

Edite seus arquivos estáticos (HTML) na pasta static, ou modifique os SCSS na pasta src. Se quiser, pode colocar imagens e arquivos JavaScript para seu projeto em assets!

Lembre-se, porém, de que este é um projeto bem básico, que explora algumas coisas superficiais do Node.js. Ainda tem muita coisa bacana para se fazer usando JavaScript no servidor e no client, portanto não pare, nem desanime! ;)
